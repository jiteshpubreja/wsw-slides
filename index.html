What is Natural Language Processing?

  • Natural Language Processing (NLP) is the use of computers to process written and spoken human language.
  • Designing and building a computer system that would analyze, understand and generate languages that humans use naturally, so that eventually you could address your computer as though you were addressing another person
  • It is a field of study which falls under the category of machine learning and more specifically computational linguistics.

Why NLTK?

  • The “Natural Language Toolkit” is a python module that provides a variety of functionality that will aide us in processing text.
  • It provides easy-to-use interfaces to over 50 corpora and lexical resources such as WordNet.
  • It provides a suite of text processing libraries for classification, tokenization, stemming, tagging, parsing, and semantic reasoning.

Installation

Open your terminal and run the following command: 

            pip install nltk
          if using Python 2
or 

            pip3 install nltk
          if using Python 3
Once completed, Run the following in your python environment: 

            import nltk
            nltk.download()
You will be presented with the following output: 

            NLTK Downloader
            ---------------------------------------------------------------------
            d) Download   l) List    u) Update   c) Config   h) Help   q) Quit
            ---------------------------------------------------------------------
            Downloader>
          enter d to download and enter 'all' as identifier.
Alternatively, You can use the following code to directly download everything: 

            import nltk
            nltk.download('all')

Chunking


              def chunker(input_words,n):
              words = input_words.split(" ")
              output = []
              current_chunk = []
              count = 0
              for w in words:
                  current_chunk.append(w)
                  count = count + 1
                  if(count == n):
                      output.append(current_chunk)
                      current_chunk = []
                      count = 0
              return output
